Abstract

Easy Read is an accessibility format for people with intellectual disabilities—but producing it requires trained specialists and doesn't scale. We built a graph-guided LLM pipeline using Abstract Meaning Representation (AMR) graphs and a Graph Neural Network (GNN) classifier to encode structural differences between standard and Easy Read text, translating graph features into LLM prompt guidance to produce more faithful, accessible simplifications.

How we evaluated

Three metrics evaluated output quality: SARI (rewards good additions, deletions, and kept words vs. a reference), BLEU (n-gram overlap with the reference Easy Read text), and Flesch-Kincaid grade level (readability). Each record was compared against a no-GNN baseline to isolate the graph-guidance contribution.
